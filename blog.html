What is the document object model (DOM)?

DOM (Document Object Model) holo ekta interface, jar maddhome apnar HTML ba XML document ke ekta tree structure e represent kora hoy. Etar maddhome JavaScript diye webpage-er content, structure, o style ke easily access o modify kora jay, jate webpage ta dynamic ebong interactive hoy.

how do you select an element from the Dom?

DOM theke element select korar jonne JavaScript e bibhinno methods use kora hoy. getElementBy(ID,TagName,ClassName) and querySelector

what is the event delegation in the context of the DOM, and why is it useful?

Event Delegation holo ekta technique, jekhane ekta parent element er upor ekta event listener set kora hoy, jar madhome tar child elements gulor events handle kora jay. Er mane, pratyek child element er jonno alada listener set na kore sudhu ekta listener use korei sob events handle kora jay.

How do you manipulate an element's attributes and styles using the Dom?

DOM er maddhome apni kono HTML element er attributes o styles manipulate korte paren.
